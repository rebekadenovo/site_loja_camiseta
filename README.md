# 🛍️ Loja de Camisetas

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

## 📌 Funcionalidades
- Exibição de produtos
- Formulário de pedido
- Vídeos demonstrativos
- Layout responsivo (mobile, tablet e desktop)

## 🛠️ Tecnologias
- HTML5
- CSS3

## 📱 Responsividade
O site foi desenvolvido utilizando:
- Grid Layout
- Media Queries

## 📁 Estrutura
- index.html
- css/style.css
- img/
- videos/

## Wireframe

O layout do projeto segue a seguinte estrutura:

- Header com navegação
- Seção de produtos em grid
- Formulário de pedido
- Seção de vídeos
- Footer

Arquivo disponível em: /assets/wireframe.png
