# 🛍️ Loja de Camisetas

Projeto desenvolvido para a disciplina de Engenharia de Interface — HTML5/CSS3.

## 📌 Funcionalidades
- Navbar responsiva com hamburger menu
- Grade de cards com variante destaque (card--featured)
- Footer com grid de colunas
- Formulário de pedido estilizado
- Vídeos demonstrativos
- Layout responsivo (mobile, tablet e desktop)

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript (vanilla)
- Google Fonts (Inter + Poppins)

## 📱 Responsividade
- Mobile: 1 coluna (< 768px) — menu hamburger ativo
- Tablet: 2 colunas (≥ 768px)
- Desktop: 3 colunas (≥ 1200px) — card destaque ocupa 2 colunas
